# sjt-tool-components

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```


# add New users (npm)
npm adduser

# login npm (npm)
npm login 
# identifying (npm)
npm whoami  

# publish (npm)
npm publish  

# delete(npm)
npm unpublish XXXX@1.0 --force
